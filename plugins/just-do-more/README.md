# Just Do More Plugin

This folder is the packaged Codex plugin distribution for Just Do More.

## Contents

- [`.codex-plugin/plugin.json`](.codex-plugin/plugin.json): plugin metadata
- [`skills/just-do-more/SKILL.md`](skills/just-do-more/SKILL.md): the skill behavior
- [`skills/just-do-more/agents/openai.yaml`](skills/just-do-more/agents/openai.yaml): Codex agent metadata
- [`assets/`](assets): plugin icon and logo assets

Use this folder when you want to distribute Just Do More as a Codex plugin. For the fastest local trial, use the raw skill path in [`../../codex/README.md`](../../codex/README.md).
