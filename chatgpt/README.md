# ChatGPT

Just Do More works in ChatGPT two ways:

- **Custom Instructions**: fastest setup, persistent across chats
- **Custom GPT**: best when you want a shareable product with its own name, description, and starters

## Files In This Folder

- [`custom-instructions.txt`](custom-instructions.txt): ready to paste into ChatGPT Custom Instructions
- [`gpt-config.md`](gpt-config.md): name, description, conversation starters, and instructions for a custom GPT

## Fastest Path

1. Open [`custom-instructions.txt`](custom-instructions.txt).
2. Paste the contents into ChatGPT Custom Instructions.
3. Try one of the prompts from [`../examples/README.md`](../examples/README.md) and compare the answer shape.

## Which Option To Use

- use **Custom Instructions** if you want the behavior for your own account quickly
- use a **Custom GPT** if you want something people can open, understand, and reuse

Custom GPT creation is done on the ChatGPT web editor and requires a plan that supports GPT building. Custom Instructions are lighter weight and easier to keep personal.

## Custom GPT Path

Use [`gpt-config.md`](gpt-config.md) as the source for the GPT name, description, conversation starters, and instruction block. Knowledge files are optional; the worked examples are the best supporting material if you want the GPT to demonstrate the product's style.
